### BL-01. Переработка шага 3.4 «Детализация стен»

**Приоритет:** высокий (UX-блокер)

**Проблема:** Шаг 3.4 «Обход периметра» непонятен пользователю — после замера длин стен (3.3) снова предлагается «обходить» те же стены. Вкладки создают ощущение произвольного доступа вместо последовательного обхода.

**Решения:**

1. **Переименование и объяснение:** Заголовок «3.4 Детализация стен». Подзаголовок: «Разбейте стену на участки: просто стена, окно, дверь, выступ. Если особых участков нет — нажмите «Далее».»
2. **Линейная навигация:** Убрать вкладки → кнопки «← Назад» / «Следующая стена →» с прогрессом «Стена 2 из 6».
3. **Предзаполнение длины остатком:** Поле «Длина, м» при открытии формы = (длина стены − сумма сегментов). Обновляется после каждого добавления.
4. **Пропуск стены:** Кнопка «Далее» работает без сегментов — стена без проёмов это валидное состояние.
5. **Мини-схема с выделенной стеной:** В шапке — SVG комнаты (из CornerLabelStep) с подсвеченным текущим пролётом.

**Статус:** реализовано
